## Added — [P0-9.1a] Year filter wiring (DONE)

`search_cnki(year_min=..., year_max=...)` now actually filters by publication
year. Recipe (validated via 6-scenario test 2026-07-15):
- `Field=PT, Operator=GT, Value=YYYY/01/01` (greater than start of year_min)
- `Field=PT, Operator=LT, Value=YYYY/12/31` (less than end of year_max)
- Format `YYYY/MM/DD` or `YYYYMMDD` confirmed working
- Operators: GT/LT work; EQ/GTE/LTE all return `非法逻辑操作符`

**Test scenarios** (test_output/_test_year_v3975.py, all PASS):
- year_min=2024, year_max=2024 → all 2024
- year_min=2020, year_max=2024 → all ≤2024
- year_max=2024 only → all 2024
- year_min=2025, year_max=2026 → all 2025-2026
- year_min=2020 only → all ≥2020
- baseline (no filter) → 345,830

## Added — [P0-9.1c] Page-2+ jitter + captcha retry (DONE)

`search_cnki(limit > 20)` pagination:
- Sleep: 1.5s fixed → `random.uniform(2000, 5000)` ms (jitter)
- On captcha: 1 retry with 30s wait (cookies may re-validate)
- Graceful degradation: if retry fails, return what we have so far
